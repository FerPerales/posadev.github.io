Call for Speakers

Para esta nueva version, tenemos una dinamica para seleccionar las charlas que estaran en la agenda oficial del evento.

Debes revisar si el tema que deseas compartir con la comundiad se encuentra entre los opciones que hemos selecionado. Luego de esto debes enviar un Pull Request (PR) al repositorio oficial de la comunidad, los PR con mayor número de likes pasaran a ser evaluados por el comite quien definirá cuales son finalmente las charlas selecionadas para la agenda.
